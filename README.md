# Git — конспект команд и приёмов

[![Git](https://img.shields.io/badge/Git-F05032?logo=git&logoColor=white)](https://git-scm.com/)
[![Language](https://img.shields.io/badge/язык-русский-blue)]()

Персональный конспект по системе контроля версий **Git**: от базовых команд до ветвления, подмодулей и практических рецептов. Собран как удобный справочник для повседневной работы.

📖 **Полный материал:** [MainGit.md](MainGit.md)

## Содержание

- **Введение** — что такое Git и как устроена распределённая модель.
- **Работа с удалённым репозиторием** — `clone`, `remote`, `push`, `pull`, `fetch`.
- **Работа с локальным репозиторием**
  - Базовые команды (`add`, `commit`, `status`, `log`, `diff`).
  - Ветвление (`branch`, `checkout`, `merge`, `rebase`).
  - Подмодули (`submodule`).
  - Прочие команды и возможности.
- **Серверные команды репозитория.**
- **Рецепты** — готовые решения типовых задач.

## Кому пригодится

Шпаргалка для тех, кто изучает Git или хочет держать под рукой структурированный справочник команд на русском языке — с оглавлением и переходами по разделам.
